# Audit log

Immutable trail of privileged actions. The Markdown view shows the same feed without styling — ideal for agents that want to scan the data.

IMPORTANT

Audit entries cannot be edited once written. Retention is 400 days.

### SCOPE

- • All mutations from the admin console.
- • System-triggered rotations and backups.
- • Report exports and data downloads.
